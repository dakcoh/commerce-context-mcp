# commerce-context-mcp

Java/Spring ecommerce backend knowledge MCP server for AI coding tools.

AI 코딩 도구에 Java/Spring 이커머스 백엔드 지식을 연결하는 MCP 서버입니다.

## Quick Start

```bash
npx commerce-context-mcp doctor
```

## MCP Config

```json
{
  "mcpServers": {
    "commerce-context": {
      "command": "npx",
      "args": ["-y", "commerce-context-mcp"]
    }
  }
}
```

## What It Covers

- Inventory overselling and reservation risks
- Payment idempotency and webhook duplication
- Refund and cancellation consistency
- Settlement reconciliation
- Coupon and promotion pitfalls
- Java/Spring implementation checklists

## Requirements

- Node.js 16+
- Java 17+
